# GAS-study-support-bot

# 📘 GAS Study Support Bot
Google Apps Script（GAS）と Gemini API を活用して構築した、学習サポート用の LINE Bot です。

ユーザーが「今日の勉強内容」「やったこと」「質問したいこと」などを LINE に送ると、  
AI が内容を解析し、  
- 学習記録の整理  
- 重要ポイントの抽出  
- 次にやるべき内容の提案  
- 過去ログの検索  
などを自動化します。

本リポジトリは **ポートフォリオ目的で構成しています**。  
AI 解析ロジックや一部の機密実装は公開していませんが、  
**全体の仕組みや設計思想が分かる構成**にしています。

---

## ✨ 主な機能

- ✍️ 学習内容の自動解析（Gemini API）
- 🗂 スプレッドシートへの学習ログ保存
- 🔍 過去データからの検索・要約
- 📊 簡易ダッシュボード表示（GAS サイドバー）
- 📱 LINE Bot との連携

## 🧩 公開範囲について

本リポジトリでは以下を **非公開** としています：

- AI 解析（Gemini）ロジック本体  
- レコメンドアルゴリズム  
- 学習要約プロンプト  
- 本番稼働中の GAS 全コード

理由：  
- 独自性のある設計のため、アイデア盗用防止  
- 実運用のため API キー等が含まれる

**代わりに以下を公開**しています：

- 参考になる GAS ハンドラの一部（安全範囲）
- 擬似コードでの全体構成
- スプレッドシート構造説明
- セットアップ手順（誰でも試作できる）
